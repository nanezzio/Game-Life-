# Курсовой проект «Игра "Жизнь"»

Консольная реализация классической клеточной игры «Жизнь», придуманной английским математиком Джоном Конвеем в 1970 году. Программа загружает начальную конфигурацию из текстового файла и пошагово отображает эволюцию вселенной.

## Правила игры

Вселенная представляет собой ограниченную плоскость, размеченную на клетки. Каждая клетка может быть \*\*живой\*\* (обозначается символом `\*`) или \*\*мёртвой\*\* (`-`). У каждой клетки есть восемь соседей.

Начальное распределение живых клеток называется \*\*первым поколением\*\*. Каждое следующее поколение рассчитывается по правилам:

1. В мёртвой клетке, рядом с которой ровно \*\*три\*\* живые клетки, зарождается жизнь.
2. Если у живой клетки есть \*\*две\*\* или \*\*три\*\* живые соседки, она продолжает жить.
3. В противном случае (меньше двух или больше трёх соседей) клетка умирает — от одиночества или перенаселённости.

Игра завершается, когда:

- на поле не остаётся ни одной живой клетки;
- очередной шаг не меняет состояние ни одной клетки (наступает стагнация).

Игрок только задаёт начальную конфигурацию и наблюдает за эволюцией.


## Требования к реализации

В программе соблюдены следующие условия:

- Текущее состояние вселенной выводится на консоль в виде таблицы символов, разделённых пробелом:
	- `\*` – живая клетка,
	- `-` – мёртвая клетка.

- На каждом шаге отображается:
- само поле,
- номер поколения,
- количество живых клеток.
- При завершении игры дополнительно выводится причина окончания.
- Расчёт соседей ведётся только в пределах поля: у граничных клеток меньше восьми соседей.
- Начальная конфигурация загружается из текстового файла.
